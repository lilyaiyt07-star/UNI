💻 The UNI OS

A simple web-based operating system made using HTML, CSS, and JavaScript. This project simulates a basic desktop environment inside a browser with multiple apps and interactive UI.

🚀 Features
📝 Notes app (auto saves using local storage)
🧮 Calculator
📅 Scrollable Calendar (month by month)
🖼️ Wallpaper changer (saved after refresh)
🎨 Paint app (draw on canvas)
📁 File viewer (upload & list files)
📷 Camera (capture using webcam)
🎧 Music player (play local songs)
🔍 Search bar for apps
🪟 Draggable & resizable windows
💾 Layout saving (positions stay after refresh)
🎯 Project Idea

The UNI OS is created to simulate how a real operating system works but in a very simple way. It helps beginners understand concepts like:

Window management
App handling
User interactions (drag, click, open, close)
Saving data in browser
🛠️ Technologies Used
HTML
CSS
JavaScript
Local Storage
📂 How to Run
Download or clone the repository
Open the project folder
Run the index.html file in your browser
